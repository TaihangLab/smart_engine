一、问题列表（7个）
================================================================================

1. JWT字段为空导致报错
   - 现象: 外部JWT只保证tenantId和userId非空，其他字段为空时系统无法运行
   - 状态: ✅ 已修复

2. tenant_id=0被当成空值
   - 现象: 字符串"000000"被转换为整数0，与模板租户冲突
   - 状态: ✅ 已修复

3. 模板租户可被分配
   - 现象: 租户0的ID可能被生产环境使用
   - 状态: ✅ 已修复

4. 外部user_id未存储
   - 现象: 综管平台的user_id没有独立字段存储
   - 状态: ✅ 已修复

5. 用户判重问题
   - 现象: 使用tenant_id+user_name判重，同名员工无法都使用
   - 状态: ✅ 已修复

6. 缺少登录页面控制
   - 现象: 没有开关控制是否跳转登录页面
   - 状态: ✅ 已修复

7. 无法配置超管
   - 现象: 所有人都是ROLE_ACCESS角色，无法配置超管ROLE_ALL
   - 状态: ✅ 已修复
