系统将自动加载根目录下的.env文件，读取其中的配置。
- docs/env.example 是一个示例文件，用于展示如何定义环境变量。

## 初始化
使用.env.example文件创建.env文件，并填写相应的值。

## 开发阶段
同时在  .env 文件 和 env.example 文件中定义变量。并提供完善的注释信息，方便开发人员理解。

## 核心配置项说明

### 数据库配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| MYSQL_SERVER | 127.0.0.1 | MySQL服务器地址 |
| MYSQL_USER | root | MySQL用户名 |
| MYSQL_PASSWORD | root | MySQL密码 |
| MYSQL_DB | smart_vision | 数据库名称 |
| MYSQL_PORT | 3306 | MySQL端口 |

### RabbitMQ 配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| RABBITMQ_ENABLED | False | 是否启用RabbitMQ消息队列 |
| RABBITMQ_HOST | 127.0.0.1 | RabbitMQ服务器地址 |
| RABBITMQ_PORT | 5672 | RabbitMQ端口 |
| RABBITMQ_USER | admin | RabbitMQ用户名 |
| RABBITMQ_PASSWORD | admin | RabbitMQ密码 |

**说明**：
- 设置 `RABBITMQ_ENABLED=False` 可完全禁用 RabbitMQ，不会启动消费者线程或尝试连接
- 禁用后，预警消息将通过 SSE 直接推送给前端，不经过消息队列

### Redis 配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| REDIS_ENABLED | False | 是否启用Redis缓存 |
| REDIS_HOST | 127.0.0.1 | Redis服务器地址 |
| REDIS_PORT | 6379 | Redis端口 |
| REDIS_DB | 0 | Redis数据库编号 |

### MinIO 配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| MINIO_ENABLED | False | 是否启用MinIO对象存储 |
| MINIO_ENDPOINT | 192.168.0.14 | MinIO服务器地址 |
| MINIO_PORT | 9000 | MinIO端口 |
| MINIO_ACCESS_KEY | minioadmin | 访问密钥 |
| MINIO_SECRET_KEY | minioadmin | 秘密密钥 |

### 缓存配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| CACHE_DEFAULT_TTL | 300 | 默认缓存时间（秒） |
| CACHE_PERMISSION_TTL | 600 | 权限缓存时间（秒） |
| CACHE_MENU_TTL | 300 | 菜单缓存时间（秒） |
| CACHE_MAX_SIZE | 128 | 最大缓存条目数 |

### 预警数据Mock配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| ALERT_MOCK_ENABLED | False | 是否启用预警数据Mock服务 |
| ALERT_MOCK_DAILY_TARGET | 50 | 每日目标预警数量 |
| ALERT_MOCK_LOOKBACK_DAYS | 8 | 回溯天数 |

**说明**：
- 用于开发和测试环境，自动填充预警数据以满足大屏展示需求
- 系统启动时自动检查并补充数据
- 仅在 `ALERT_MOCK_ENABLED=True` 时生效

## 服务开关说明

通过环境变量可以控制各个服务的启动：

| 服务开关 | 默认值 | 控制内容 |
|----------|--------|----------|
| SYSTEM_CORE_ENABLED | True | 系统核心服务 |
| NACOS_ENABLED | False | Nacos服务注册 |
| MINIO_ENABLED | False | MinIO服务集群 |
| COMPENSATION_AUTO_START | False | 补偿服务自动启动 |
| TRITON_SYNC_ENABLED | False | Triton模型同步 |
| SKILL_MANAGER_ENABLED | False | 技能管理器 |
| AI_TASK_EXECUTOR_ENABLED | False | AI任务执行器 |
| SSE_MANAGER_ENABLED | False | SSE连接管理器 |
| REDIS_ENABLED | False | Redis服务 |
| LLM_TASK_EXECUTOR_ENABLED | False | LLM任务执行器 |
| ALERT_REVIEW_QUEUE_ENABLED | True | 预警复判队列 |
| RABBITMQ_ENABLED | False | RabbitMQ消息队列 |
| ALERT_MOCK_ENABLED | False | 预警数据Mock服务 |

**开发环境建议配置**：
- 仅启用必要的服务（如数据库、SSE）
- 禁用外部依赖服务（如 RabbitMQ、Redis、MinIO）
- 使用 Mock 服务进行测试（如 ALERT_MOCK_ENABLED）
